Introduces a read-only command that serves the field reference for each graph element (node, aspect, architecture, config, flow). The references are embedded in the CLI binary and resolve without loading a project graph, so every reader — agent or human — consults a single source that travels with the tool, rather than per-project copies of schema files that can drift apart between projects or go stale after an upgrade. The command mirrors the existing knowledge-browser surface: a list subcommand and a read subcommand that prints one schema's reference, with an unknown name reported as a clear what/why/next error.
