## [2026-05-28T12:19:48.059Z]
Two changes to buildTestGraph for validator unit-test ergonomics. First, the default rootPath was '/tmp/test-graph' which made path.dirname(rootPath)='/tmp' — and the validator's stage-4 file-system passes (walkRepoFiles, mapping coverage checks, secrets inspection) iterate from that project root. Iterating all of /tmp timed out tests. The helper now allocates an isolated mkdtemp prefix per call when rootPath is not supplied, with rootPath set to a .yggdrasil subdirectory inside it; consumers that need a specific rootPath can still pass one. Second, schemas: [] is now part of the returned Graph because validator.checkSchemas reads graph.schemas.map(); without the property defined, that call throws at module-test setup time. Both adjustments are test-helper concerns only and do not alter production behavior.
