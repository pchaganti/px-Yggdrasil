Introduced this module by extracting the drift-cause helpers (identity diff plus the per-cause human-facing description/token/layer functions) out of the check engine and the core graph-query layer. The motivation was a per-node reviewer-context budget: both source nodes had grown past the budget after the drift baseline gained a typed upstream-identity block, and the reviewer prompt for an oversized node risks context-window truncation that falsely rejects unchanged code. These helpers are pure transforms over already-loaded graph and identity data, so they form a cohesive standalone unit with no behavior change.
