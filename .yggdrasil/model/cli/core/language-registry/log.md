## [2026-05-31T13:43:41.297Z]
This registry is the single place that maps a source-file extension to the tree-sitter grammar that parses it and to the comment node-type names and delimiters that comment-based deterministic rules and inline suppression markers rely on. It was expanded from the TypeScript and JavaScript family to also cover Python, Go, Rust, Java, C#, C, C++, PHP, Ruby, and JSON so deterministic checks can reason about those languages out of the box. The comment metadata for each language was determined by parsing a real sample with that grammar rather than assumed, because a wrong value would silently disable comment-based rules and suppression for that language while leaving structural rules working. It lives in its own unit so the grouping of graph-query helpers it formerly shared a unit with stays within the per-unit review-context size limit.
