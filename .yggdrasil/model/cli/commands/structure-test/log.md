## [2026-05-29T08:25:58.552Z]
Implements the yg structure-test command as a mirror of yg ast-test. The command runs a structure aspect (reviewer.type: structure) against a named graph node without recording a baseline — useful for iterating on check.mjs during development. The --check-determinism flag runs the check twice and exits with code 2 if results differ, detecting non-stable check functions before they cause flaky CI. The command follows the same error-reporting convention as all CLI commands: buildIssueMessage for every user-facing error, non-zero exit on violations.
