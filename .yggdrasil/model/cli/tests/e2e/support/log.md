Introduces a shared helper that lets the end-to-end suites exercise the language-model reviewer path without any real model. The reviewer normally talks to an external model over HTTP, which is slow and non-deterministic; this helper stands up a tiny local stand-in that speaks the same protocol on a freshly-assigned loopback port and records every request, so a test can dictate the verdict and then assert on what the system actually sent and how it processed the reply. A second helper spawns the binary asynchronously, which is required because the stand-in and the binary run in the same test process — a synchronous spawn would freeze the loop that must answer the binary's calls. The motivation is that the most-used reviewer type was previously only testable against a live model or a dead endpoint, leaving its real mechanics unproven in continuous integration; this helper makes that whole path deterministic and CI-runnable.
